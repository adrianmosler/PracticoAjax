Ejercicio 4

Realizar un formulario web para cargar contactos. Al presionar el botón "Enviar" se deberán validar los datos en cliente. Si algún campo es erróneo, deben mostrar el campo en color rojo y se muestra el mensaje de error correspondiente. Si no existen errores, enviar los datos al servidor donde se validará los datos una vez más y se cargarán en la base de datos. Si todo ha salido bien en servidor, se envía a cliente la cadena "Ok"; caso contrario se enviará un "Error". Para finalizar, el cliente deberá mostrar un mensaje de error o confirmación según corresponda
